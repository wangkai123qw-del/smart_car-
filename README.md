Smart_Car 智能小车
项目简介
基于 STM32F103C8T6 的自主移动小车，集成红外避障与循迹功能，通过传感器数据实时决策行驶路径，实现嵌入式控制系统从感知到执行的完整闭环。

技术栈
MCU：STM32F103C8T6

传感器：TCRT5000 红外反射传感器 × 3（避障×1 + 循迹×2）

执行器：直流电机 × 2、L298N 电机驱动模块

开发环境：Keil MDK + ST-Link

调试工具：串口监视器

编程语言：C

核心功能
自动避障
前置红外传感器检测障碍物反射光强度

反射光低于阈值时判定存在障碍物，触发避障决策

执行「后退 → 转向」动作序列，重新规划行进方向

自主循迹
底部双路红外传感器检测地面黑线与白板的光反射差异

左右传感器独立判断偏移方向：

左偏 → 右轮加速，修正轨迹

右偏 → 左轮加速，修正轨迹

实时 PID 增量式调节 PWM 占空比，实现平滑循线

硬件平台
模块	型号	功能
主控	STM32F103C8T6	核心控制与决策
红外传感器	TCRT5000 × 3	避障检测 ×1 / 循迹检测 ×2
电机驱动	L298N	双路直流电机驱动
执行器	直流电机 × 2	后轮差速驱动
调试	ST-Link / UART	下载调试与日志输出
关键实现
传感器数据采集：通过 ADC 读取 TCRT5000 模拟电压值，软件阈值滤波消除环境光干扰

电机控制：使用 TIM 输出双路 PWM 独立调速，GPIO 控制 L298N 实现正反转

控制逻辑：避障与循迹状态机切换，优先级避障 > 循迹，确保安全性

调试支持：串口打印传感器数据与运行状态，辅助参数整定

项目亮点
基于 STM32F103 完成传感器驱动 → 逻辑决策 → 执行控制的完整嵌入式开发流程

实现红外传感器数据采集与软件滤波处理

多模式切换：避障模式 / 循迹模式，支持独立运行
